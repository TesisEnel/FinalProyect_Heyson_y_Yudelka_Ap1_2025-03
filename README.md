Descripción General del Proyecto (Mini Manual)

Este sistema permite gestionar los procesos administrativos de una Tesorería Municipal.
Los usuarios pueden registrar solicitantes, emitir procesos, generar recibos, subir documentos obligatorios y consultar historial de ingresos.

Incluye:
	•	Gestión completa de Derecho de Enterramiento
	•	Gestión de Derecho de Construcción
	•	Gestión de Registro de Documentación
	•	Gestión de Certificaciones
	•	Gestión de Arrendamiento de Terrenos
	•	Generación automática de Recibos de Ingresos
	•	Subida de archivos obligatorios (cédula, acta, recibos, fotos, etc.)
	•	Consultas filtradas por rangos de fechas
	•	Reportes en PDF y Excel
	•	Auditoría básica mediante historial
	•	Roles: Admin y Tesorera
	•	Validaciones estrictas en formularios
	•	Incorporación de documentos al proceso antes de permitir generar el recibo
